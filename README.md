# 💎 Crystal – Web3 Super App Landing Page

> The bridge between traditional finance and Web3. A cyber-black & neon green metaverse-inspired landing page with Fiat + Crypto wallet features, AI assistant, virtual cards, newsletter signup, and Web3 login mock.

## 🚀 Live Preview
Open `index.html` in any modern browser.

## 📁 Project Structure
- index.html — main landing page
- CSS/styles.css — styling (note: folder name is `CSS`)
- js/main.js — UI interactions and demo modal logic
- js/newsletter.js — newsletter subscription (mocked via localStorage)
- README.md — this file

## 🛠 Local preview
1. Clone the repo:
   git clone https://github.com/57KingVibe/crystal-web3-super-bank.git
2. Open `index.html` in a browser (or serve with a simple static server like `npx http-server`).

## ⚠️ Notes
- The demo newsletter uses localStorage as a mock. No backend required.
- There is a duplicate folder named `Crystal Web3 super bank` that appears to contain copies of the same files; confirm if you want those removed.
